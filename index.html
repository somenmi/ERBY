О программе

ERBY (Easy Roadmap Builder of Yalkee) - визуальный конструктор дорожных карт для обучения и планирования. Создавайте контейнеры, устанавливайте связи и отслеживайте прогресс.

Горячие клавиши

  • F1 - Открыть эту справку
  • Esc - Отмена
  • Delete - Удалить выбранный контейнер
  • L - Блокировка/разблокировка ВСЕХ окон
  • C - Режим создания связей
  • S - Сохранить в браузере
  • Ctrl + S - Экспорт в JSON
  • N - Открыть/закрыть блокнот
  • M - Управление roadmap'ами

Управление мышью

  • ЛКМ на контейнер + перетаскивание - перемещение (+ выделение связей)
  • ПКМ на контейнер - блокировка/разблокировка окна
  • Клик на квадраты прогресса - отметить прогресс (12 уровней)
  • Двойной клик на описание контейнера - редактирование
  • Режим связей: клик на 1-й контейнер, затем на 2-й
  • Клик на линию связи - удалить связь (выделяется красным)

Управление Roadmap's

Сохранение данных

  • Автосохранение - при любых изменениях
  • LocalStorage - данные хранятся в вашем браузере
  • JSON экспорт/импорт - для резервных копий

Блокнот

  • Автосохранение - все изменения сохраняются автоматически
  • Форматирование - используйте панель инструментов для форматирования текста
  • Экспорт - можно экспортировать заметки отдельным файлом
  • Статистика - отслеживайте количество символов, строк и слов
  • Горячие клавиши в блокноте: Ctrl+B (жирный), Ctrl+I (курсив), Ctrl+U (подчёркивание), Ctrl+S (зачёркивание)
  • Ссылки - для добавления используйте кнопку ссылки или Ctrl+K
  • Клик по ссылкам - зажмите Ctrl и кликните по ссылке для открытия (ссылки не кликабельны без Ctrl для предотвращения случайных кликов)

Система множественных Roadmap'ов (сокращенно - RM)

  • Несколько RM'ов - создавайте отдельные RM'ы с уникальными URL
  • URL формат: https://somenmi.github.io/ERBY/ - дефолтный
  • URL формат: https://somenmi.github.io/ERBY/my-project123 - RM "my-project123"
  • Переключение - используйте окно управления (кнопка или клавиша M)
  • Автосохранение - каждый RM сохраняется отдельно только в вашем браузере
  • Экспорт/импорт - каждый RM можно экспортировать отдельно
  • Удаление - можно удалять неактивные RM'ы из списка

Важно

  • Нельзя удалить активный roadmap - сначала переключитесь на другой
  • Все данные хранятся локально в браузере `Перейти на свой уникальный Roadmap с другого устройства, по вашему локальному URL — НЕВОЗМОЖНО`
  • Для переноса данных используйте экспорт/импорт
