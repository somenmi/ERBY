## Особенности
- 📝 Создание и редактирование контейнеров
- 🔗 Установка связей между этапами
- 🔒 Гибкая система блокировки
- 📊 Отслеживание прогресса (12 уровней)
- 💾 Автосохранение в браузере
- 📄 Импорт/экспорт в JSON
- 📓 Встроенный сохраняемый блокнот

# Версии
### v1.0.0 : Релиз
    × Конструктор дорожных карт с интуитивным созданием контейнеров
    × Система связей между элементами с визуальным выделением при перетаскивании
    × Экспорт/импорт проектов в JSON (вся структура + заметки)
    × Встроенный блокнот для дополнительных записей и пометок
    × Отдельный экспорт заметок в текстовом формате
    × Прогресс-трекер с 12-уровневой системой отслеживания
    × Блокировка элементов (отдельных или всех сразу)
    × Горячие клавиши для быстрого доступа к основным функциям
    × Стартовые шаблоны для быстрого начала работы

    × Удобства
        ↪︎ Сохранение в браузере (автосохранение при изменениях)
        ↪︎ Гибкое перетаскивание контейнеров
        ↪︎ Контекстное меню для блокировки
        ↪︎ Всплывающие подсказки
